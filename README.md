# TraceChain - Blockchain Based Food Traceability System

## Overview

TraceChain is a full-stack blockchain-powered food traceability platform designed to monitor and verify the lifecycle of food products across the supply chain.

The system integrates blockchain verification, IoT telemetry monitoring, AI-powered assistance, multilingual support, and real-time logistics tracking to ensure transparency, safety, and integrity in food transportation and storage.

---

## Features

- Blockchain-based supply chain verification
- Real-time product traceability
- Interactive logistics maps and journey timeline
- AI assistant with voice support
- Role-based dashboards
- Temperature and humidity monitoring
- Alert and anomaly detection
- Multilingual support
- Ethereum smart contract integration
- MongoDB-based data persistence

## Installation

### Clone Repository

```bash
git clone https://github.com/sh2900/FoodTraceabilityProject.git
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd ../backend_node
npm install
```

### Run Frontend

```bash
npm start
```

### Run Backend

```bash
node index.js
```
